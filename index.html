pertama kita buat folder berkas, lalu buka gitbash dan ketikan perintah git init untuk mengecek folder setelah itu kita ketikan perintah git status untuk mengecek file yang sudah kita buat

baru kita buat perintah git add . untuk menabhkan git ke terminal

setelah menambahkan git kita buat commit (sebuah deklarasi untuk file yang kita tambahkan ). baru setelah itu kita buat new reposity di github lalu salin link nek repsity itu dan pastekan pada perintah git remote add origin url tambhakn url new reposity tadi

baru kita submit ke github dengan cara membuat perintah di git bash git push -u origin master
